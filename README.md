## About

Personal portfolio of **Surya Prakash Singh** — a web developer and co-founder of [Team Tech Pro](https://teamtechpro.netlify.app). Built from scratch with vanilla HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean, fast, handcrafted code.

---

## Features

- **Dark / Light mode** — persisted via `localStorage`
- **Typing animation** — sequential hero text reveal
- **Smooth scroll navigation** — active link highlighting on scroll
- **Glassmorphism UI** — frosted header, card depth, gradient accents
- **Spinning avatar ring** — CSS conic gradient animation
- **Floating badges** — animated hero visual elements
- **Contact form** — powered by [Web3Forms](https://web3forms.com) API
- **Lucide icons** — locally hosted, no CDN dependency
- **Fully responsive** — mobile, tablet, desktop

## Getting Started

No build step needed — just open in a browser.

```bash
git clone https://github.com/suryaprakash9695/suryaprakashportfolio.git
cd suryaprakashportfolio
# Open index.html in your browser
```

Or serve locally:

```bash
npx serve .
# → http://localhost:3000
```
